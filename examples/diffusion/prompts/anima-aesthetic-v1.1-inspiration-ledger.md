# Anima Aesthetic 1.1 calibration inspiration ledger

This is a hand-authored ideation ledger, not a prompt template bank. Titles and
creator names never appear in the calibration prompts. Each note isolates a
visual or narrative ingredient that can be recombined by judgment, not code.
