# Anima Aesthetic 1.1 visual-artist diversity ledger

This hand-authored ledger records visual-language references, not prompt text.
The final calibration captions must never contain artist names or request direct
imitation. Cross-references use only abstract properties such as line, shape,
palette, texture, space, costume construction, and compositional rhythm.
